# SiteSafe RAMS

Next.js + Supabase starter for site-specific RAMS access, collapsible sections and operative sign-off.

## Vercel Environment Variables

Add these in Vercel:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
